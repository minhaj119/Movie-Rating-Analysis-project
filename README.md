Analyzed a movie dataset using Python to find trends between IMDb ratings, genres, and release years. Visualized patterns to identify how movie type and year impact audience ratings and box office performance.
# Movie Rating Analysis

This project analyzes the relationship between **IMDb ratings**, **movie genres**, and **box office performance** using Python libraries such as Pandas, Matplotlib, and Seaborn.

## Objective
To explore how movie genres and release years impact IMDb ratings and identify patterns in audience preferences.

## Key Insights
- Sci-Fi and Thriller movies tend to have the highest average IMDb ratings.
- Box office success doesn't always correlate directly with IMDb ratings.
- Visualization of rating trends helps understand viewer interests over time.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook / Google Colab

## Steps Performed
1. Created a sample dataset of movie details.
2. Analyzed IMDb rating statistics by genre.
3. Visualized correlations between Box Office and IMDb ratings.
4. Exported data and findings to a CSV file.

##  Results
Clear insights were drawn showing audience preferences and how different movie genres perform both critically and commercially.
